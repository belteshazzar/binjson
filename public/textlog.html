Versioned Text Log (WebAssembly) with OPFS

This page demonstrates the WebAssembly text log (TextLog from binjson-wasm.js) — an append-only history of a document. Each add records a full version; the log keeps periodic snapshots plus diffs so any past version can be reconstructed and compared. It runs in a Web Worker.

How it's used

import { TextLog } from '@belteshazzar/binjson/wasm';

const log = new TextLog(syncHandle, /* diffsPerSnapshot */ 10);
await log.open();

const v = await log.addVersion('the quick brown fox');  // -> version number
const text = await log.getVersion(1);       // full text at a version
const diff = await log.getDiff(1, 2);       // human-readable unified diff
const hash = await log.getVersionHash(2);   // SHA-256 hex
log.getCurrentVersion();                    // latest version number
await log.close();
