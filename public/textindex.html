Full-Text Index (WebAssembly) with OPFS

This page demonstrates the WebAssembly full-text index (TextIndex from binjson-wasm.js). It tokenizes, stems, and BM25-ranks documents, stored across three OPFS B+ tree files (<name>-terms.bj, -documents.bj, -lengths.bj). All operations run in a Web Worker.

How it's used

import { TextIndex, BPlusTree } from '@belteshazzar/binjson/wasm';

// three B+ trees, one per file, share a base name
const index = new TextIndex({ trees: { index, documentTerms, documentLengths } });
await index.open();

await index.add('doc1', 'the quick brown fox');   // index a document
const ranked = await index.query('quick brown');  // [{ id, score }] BM25
const ids    = await index.query('quick', { scored: false });  // just ids
const both   = await index.query('quick brown', { requireAll: true }); // AND
await index.remove('doc1');
await index.getDocumentCount();
await index.close();
