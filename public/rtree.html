R-Tree (WebAssembly) with OPFS

This page demonstrates the WebAssembly R-tree (RTree from binjson-wasm.js) — a durable, append-only spatial index of (lat, lng) points, each tagged with a 24-hex ObjectId. It answers bounding-box and radius queries. All operations run in a Web Worker (public/worker.js); this page sends it messages.

How it's used

import { RTree } from '@belteshazzar/binjson/wasm';
import { ObjectId } from '@belteshazzar/binjson';

const tree = new RTree(syncHandle, /* maxEntries */ 9);
await tree.open();

tree.insert(40.7128, -74.0060, new ObjectId());  // point + id
const near = tree.searchRadius(40.71, -74.0, 500);  // {objectId,lat,lng,distance} within km
const box  = tree.searchBBox({ minLat: 30, maxLat: 45, minLng: -80, maxLng: -70 });
tree.remove(objectId);
tree.size();
await tree.compact(destSyncHandle);
await tree.close();

Tree file

Latitude -90..90, longitude -180..180. Leave the ObjectId blank on insert to auto-generate one.
